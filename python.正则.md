## 多行匹配

当字符串有换行符时，正则的 `.` 不会匹配换行符。有以下方法处理：
- 使用 `re.DOTALL` 标志（推荐）。
- 显示匹配换行符，在 pattern 添加 `\n` 或 `\s`。
- 显示匹配所有字符。比如 `[\s\S]*` 匹配所有空白（包括换行、制表符、空白字符）和非空白字符。

注意，re.M 影响的是 `^$` 这两个元字符。
